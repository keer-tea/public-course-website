为什么学 java ​

我其实不想学 java，虽然我对后端其实有一点兴趣，但是因为懒，感觉学起来会很累，所以一直没学。

后来，遇到了一些人和事，感觉还是有必要学习下后端。node 学习资料过少，java 学习资料多，并且周围很多 java 大佬，所以就决定学 java 了。

当然，我学 java 并不是单纯为了去公司写 java 代码，也有可能是为了证明自己，可能是我的兴趣，可能是我想了解后端。
